# Driver Drowsiness & Attention Monitoring System

A computer vision system that monitors driver alertness using facial landmarks
and temporal eye-closure analysis from a live webcam feed.

## Overview
This project detects signs of driver drowsiness such as prolonged eye closure
and reduced blink rate using classical computer vision techniques.

- OpenCV
